# Arduino-ButtonLed
Sketch Arduino che permette di accendere un led tramite un tasto, utilizzando un Arduino Uno.

# Funzionamento circuito
Esercitando pressione sul tasto il led si accenderà, rilasciando la pressione il led si spegnerà

# Materiale Utilizzato
* Scheda Arduino
* Breadboard
* Cavi Jumper
* 1 Led
* 1 tasto
* 2 resistenze 220ohm
